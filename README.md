## 📚 Modules

### 🔢 [NumPy](./NumPy) — ✅ Complete

A 5-phase roadmap covering the full NumPy toolkit, from basic array creation to image processing and linear algebra.

| Phase | File | Topics |
|---|---|---|
| 1 | `phase1_array_and_basis` | `ndarray`, `ndim`, `shape`, data types |
| 2 | `phase2_array_operations` | Arithmetic, slicing, indexing, iteration |
| 3 | `phase3_advance_operations` | Broadcasting, reshape, boolean masking |
| 4 | `phase4_image_color_reverse` | Images as 3D arrays, RGB channel manipulation |
| 5 | `phase5_statistics_matrix_vectors` | Dot products, matrix multiplication, std, variance |

**Key Skills:** Vectorization · Broadcasting · Image Processing · Linear Algebra

---

### 🐼 [Pandas](./Pandas) — ✅ Complete

A 6-notebook series on data manipulation using the Stack Overflow Developer Survey 2024 as a real dataset.

| Notebook | Topics |
|---|---|
| `Pandas_1_basics` | Series, DataFrames, reading CSV, basic exploration |
| `Pandas_2_dictionary` | Creating DataFrames from dictionaries, columns |
| `Pandas_3_indexing_and_operations` | `.loc`, `.iloc`, conditional selection, operations |
| `Pandas_4_filtering` | Boolean filters, `isin()`, `str` accessor methods |
| `Pandas_5_modifying_data` | Updating values, applying functions, type casting |
| `Pandas_6_add_remove_rows_columns` | `concat()`, `drop()`, `insert()`, axis operations |

**Dataset:** Stack Overflow Developer Survey 2024 (`data_2024/`)  
**Key Skills:** Data Cleaning · Indexing · Filtering · DataFrame Manipulation

## 🚀 How to Run

```bash
# Install dependencies
pip install numpy pandas jupyter

# Launch Jupyter
jupyter notebook
```

Then open any `.ipynb` file from the module folder.

> **Note:** The `numpy logo.npy` file in the NumPy folder is required for Phase 4 image exercises. Keep it in the same directory as the notebooks.
